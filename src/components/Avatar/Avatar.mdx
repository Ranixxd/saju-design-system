# Avatar

사용자 또는 엔티티를 시각적으로 표현하는 프로필 이미지 컴포넌트입니다.


## 표시 우선순위

1. `src` 이미지가 있으면 이미지 표시
2. 이미지가 없거나 로드 실패 시 → `name`으로 이니셜 생성
3. `name`도 없으면 → 기본 아이콘 표시


## ✅ Do

- `alt`는 이미지의 맥락을 설명하세요. (예: "홍길동 프로필 사진")
- 이니셜 모드에서는 `name`을 항상 제공하세요.
- 여러 아바타를 겹쳐 표시할 때(Avatar Stack)는 z-index와 마진을 일관되게 적용하세요.
- 클릭 가능한 아바타에는 `role="button"`과 `aria-label`을 추가하세요.

## ❌ Don't

- 장식용이 아닌 경우 `alt`를 빈 문자열로 두지 마세요.
- 아바타를 너무 작게 사용하면 이니셜이 읽기 어려워요. `xs` 이하에서는 이니셜 생략을 고려하세요.
- 그룹 아바타(Stack)에서 너무 많은 수(5개 이상)를 나열하지 마세요. "+N" 처리를 사용하세요.


## Size 선택 기준

| Size | 크기 | 사용 맥락 |
|------|------|----------|
| `xs` | 20px | 코멘트, 인라인 멘션 |
| `sm` | 28px | 목록 항목, 테이블 |
| `md` | 36px | 기본값, 카드 헤더 |
| `lg` | 48px | 프로필 섹션 |
| `xl` | 64px | 프로필 페이지, 상세 화면 |


## 접근성

- `src`가 있을 때 `alt`를 반드시 제공하세요.
- 이니셜 아바타의 배경색은 `name` 해시 기반으로 자동 생성되며 대비를 확인하세요.
- 장식용 아바타는 `alt=""`로 스크린리더에서 건너뜁니다.
