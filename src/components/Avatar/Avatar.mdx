# Avatar

사용자 또는 엔티티를 시각적으로 표현하는 요소입니다.

---

## ✅ Do

- 이미지가 없을 경우 `name`을 제공해 이니셜 폴백이 자동으로 표시되게 하세요.
- `name`에서 색상이 자동 생성되므로 동일한 사용자는 항상 같은 색상을 갖습니다.
- 여러 명을 표시할 때는 Avatar Group으로 겹쳐서 표현하세요.
- `alt` 또는 `name`으로 반드시 접근 가능한 이름을 제공하세요.

## ❌ Don't

- 이미지 URL만 제공하고 `name`이나 `alt`를 생략하지 마세요.
- 같은 목록에서 크기를 혼용하지 마세요. 일관된 `size`를 사용하세요.
- 텍스트 없이 Avatar만으로 사람 정보를 전달하지 마세요. 이름이나 역할을 함께 표시하세요.

---

## 사이즈 선택 기준

| Size | 사용 상황 |
|------|----------|
| `xs` (24px) | 댓글 목록, 인라인 멘션 |
| `sm` (32px) | 테이블 행, 콤팩트 목록 |
| `md` (40px) | 기본값, 헤더, 카드 |
| `lg` (48px) | 프로필 섹션 |
| `xl` (64px) | 프로필 페이지, 설정 페이지 |

---

## Avatar Group 사용 시

- 표시할 최대 개수를 정하고 나머지는 "+N" 형태로 표시하세요.
- 겹치는 순서는 왼쪽이 위로 오도록 `z-index`를 조정하세요.
- 호버 시 전체 목록을 Tooltip으로 보여주는 것을 고려하세요.

---

## 접근성

- `role="img"`와 `aria-label`이 자동 설정됩니다.
- 이미지 로드 실패 시 이니셜 폴백으로 자동 전환됩니다.
