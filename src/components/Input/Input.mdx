# Input

텍스트 기반 단일 줄 입력 필드입니다.


## ✅ Do

- `label`은 항상 제공하세요. 플레이스홀더는 label을 대체할 수 없습니다.
- 유효성 오류는 `isError` + `errorMessage`를 함께 사용하세요.
- 성공 상태는 `isSuccess`로 사용자에게 즉각 피드백을 주세요.
- `helperText`로 입력 형식 힌트를 미리 제공하세요. (예: "영문, 숫자 조합 8자 이상")
- 아이콘·단위 등 부가 요소는 `leftAddon` / `rightAddon`을 활용하세요.

## ❌ Don't

- `placeholder`만으로 label을 대신하지 마세요. (접근성 문제)
- 오류 메시지를 `helperText`에 넣지 마세요. `errorMessage`를 따로 사용하세요.
- 긴 텍스트 입력(3줄 이상)에는 Input 대신 Textarea를 사용하세요.
- `isError`와 `isSuccess`를 동시에 사용하지 마세요.


## Size 선택 기준

| Size | 사용 맥락 |
|------|----------|
| `sm` | 테이블 필터, 인라인 편집 |
| `md` | 기본값, 폼 필드 |
| `lg` | 검색창, 중요도 높은 단독 입력 |


## 접근성

- `label`을 제공하면 `htmlFor`와 `id`가 자동으로 연결됩니다.
- `errorMessage`가 있으면 `aria-invalid="true"`와 `aria-describedby`가 자동 설정됩니다.
- `isSuccess` 상태는 색상만으로 표현하지 말고 아이콘이나 텍스트를 함께 사용하세요.
