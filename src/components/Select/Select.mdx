# Select

미리 정해진 옵션 중 하나를 선택할 때 사용합니다.

---

## ✅ Do

- 선택 항목이 **5개 이상**일 때 Select를 사용하세요.
- `placeholder`로 "선택하세요"처럼 액션을 유도하는 텍스트를 넣으세요.
- 선택 항목에 논리적인 순서를 부여하세요. (가나다순, 빈도순, 중요도순)
- 비활성 옵션은 텍스트에 이유를 힌트로 표시하세요. (예: "준비 중")

## ❌ Don't

- 항목이 2~4개라면 RadioGroup을 사용하세요. Select는 클릭을 두 번 해야 합니다.
- `label` 없이 사용하지 마세요.
- 항목 텍스트를 20자 이상으로 길게 쓰지 마세요. 드롭다운에서 잘립니다.

---

## Select vs RadioGroup 선택 기준

| 항목 수 | 권장 컴포넌트 |
|--------|-------------|
| 2~4개 | RadioGroup |
| 5개 이상 | Select |
| 10개 이상 + 검색 필요 | Combobox (추후 구현) |

---

## 접근성

- 네이티브 `<select>` 기반이라 키보드 접근성이 기본 보장됩니다.
- 에러 시 `aria-invalid`가 자동 설정됩니다.
