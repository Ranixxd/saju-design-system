# Select

미리 정의된 옵션 중 하나를 선택할 때 사용합니다.


## ✅ Do

- 선택 가능한 옵션이 **5개 이상**일 때 사용하세요. 그 이하라면 RadioGroup을 고려하세요.
- `placeholder`로 "선택하세요" 같은 안내 문구를 제공하세요.
- `label`은 항상 제공하세요.
- 옵션 수가 많으면 관련 옵션끼리 `optgroup`으로 묶는 걸 고려하세요.

## ❌ Don't

- 2~4개의 상호 배타적 선택에는 Radio를 사용하세요.
- On/Off 토글 목적에는 Toggle을 사용하세요.
- 옵션에 이미지나 복잡한 레이아웃이 필요하면 커스텀 드롭다운을 구현하세요.
- `disabled` 옵션만 남기지 마세요.


## Size 선택 기준

| Size | 사용 맥락 |
|------|----------|
| `sm` | 테이블 필터, 인라인 설정 |
| `md` | 기본값, 폼 필드 |
| `lg` | 중요도 높은 단독 선택 |


## 접근성

- `label`을 제공하면 `htmlFor`와 `id`가 자동으로 연결됩니다.
- `errorMessage`가 있으면 `aria-invalid="true"`가 자동 설정됩니다.
- 키보드로 화살표 키를 통해 옵션을 탐색할 수 있어야 합니다.
