# Radio / RadioGroup

여러 옵션 중 **하나만 선택**할 수 있는 폼 요소입니다.

---

## ✅ Do

- 선택지가 **2~4개**일 때 사용하세요. 한눈에 모든 옵션을 비교할 수 있습니다.
- 반드시 `RadioGroup`으로 묶어 `name`을 공유하세요. 단독 Radio는 그룹 맥락 없이 의미가 없습니다.
- 옵션 중 가장 일반적인 값을 기본 선택 상태로 설정하세요.
- 옵션 텍스트는 서로 명확히 구별되어야 합니다.

## ❌ Don't

- 선택지가 5개 이상이면 Select를 사용하세요. 라디오 버튼이 많으면 스크롤이 길어집니다.
- 선택지 없이(미선택 상태) 폼을 제출할 수 있게 두지 마세요. 기본값을 설정하거나 필수임을 안내하세요.
- 라디오 버튼을 하나만 단독으로 사용하지 마세요. Checkbox를 사용하세요.

---

## 항목 수별 권장 컴포넌트

| 항목 수 | 권장 컴포넌트 |
|--------|-------------|
| 2~4개 | RadioGroup |
| 5개 이상 | Select |
| On/Off 1개 | Checkbox 또는 Toggle |

---

## 접근성

- `RadioGroup`은 `<fieldset>` + `<legend>` 구조로 그룹 레이블이 스크린리더에 전달됩니다.
- 키보드 방향키(↑↓)로 라디오 간 이동이 가능합니다.
