# Radio

상호 배타적인 옵션 중 하나를 선택할 때 사용합니다.


## ✅ Do

- 선택지가 **2~5개**이고 상호 배타적일 때 사용하세요.
- 반드시 `name` 속성으로 같은 그룹임을 명시하세요.
- 기본 선택값(default)을 제공하는 것이 좋습니다.
- 선택지가 길면 세로로, 짧으면 가로로 배치하세요.
- `fieldset`과 `legend`로 그룹을 감싸 관계를 명확히 하세요.

## ❌ Don't

- 선택지가 6개 이상이면 Select를 사용하세요.
- 여러 항목을 동시에 선택해야 한다면 Checkbox를 사용하세요.
- Radio 버튼을 선택 즉시 폼을 제출하는 용도로 사용하지 마세요.
- `label` 없이 사용하지 마세요.


## Radio vs Checkbox vs Select

| 상황 | 권장 컴포넌트 |
|------|-------------|
| 1개만 선택 (2~5개 옵션) | Radio |
| 여러 개 선택 가능 | Checkbox |
| 1개만 선택 (6개 이상 옵션) | Select |
| 즉시 반영되는 On/Off | Toggle |


## 접근성

- `label`을 제공하면 `htmlFor`와 `id`가 자동으로 연결됩니다.
- `isError` 상태에서 `aria-invalid="true"`가 자동 설정됩니다.
- 그룹 전체에 `role="radiogroup"`과 `aria-labelledby`를 제공하세요.
- 키보드 화살표 키로 그룹 내 탐색이 가능해야 합니다.
