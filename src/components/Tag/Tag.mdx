# Tag

사용자가 추가하거나 제거할 수 있는 분류/필터 레이블입니다.


## ✅ Do

- 사용자가 직접 **추가하거나 제거**할 수 있는 항목에 사용하세요.
- 검색 필터, 키워드, 선택된 카테고리 표시에 적합합니다.
- 제거 가능한 태그에는 `onRemove`를 제공하세요.
- 여러 태그는 가로로 나열하고 줄 바꿈을 허용하세요.

## ❌ Don't

- 상태 표시 목적이라면 Tag 대신 Badge를 사용하세요.
- 태그 텍스트가 너무 길면 잘라내거나 툴팁을 제공하세요.
- 클릭으로 선택/해제되는 필터라면 별도의 선택 상태 스타일이 필요합니다.


## Variant 사용 기준

| Variant | 사용 예 |
|---------|--------|
| `default` | 일반 태그, 키워드 |
| `primary` | 선택된 필터, 활성 태그 |
| `success` | 완료된 항목 |
| `warning` | 주의 필요 항목 |
| `danger` | 오류 또는 삭제 대상 |


## 접근성

- `onRemove`가 있으면 제거 버튼에 `aria-label="Remove tag"`가 자동 설정됩니다.
- 키보드로 제거 버튼에 포커스하고 Enter/Space로 제거할 수 있어야 합니다.
- 태그 목록 전체는 `role="list"`로 감싸는 것을 권장합니다.
