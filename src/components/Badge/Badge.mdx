# Badge

상태, 카테고리, 수량 등을 시각적으로 강조하는 작은 레이블입니다.


## Variant 사용 기준

| Variant | 의미 | 사용 예 |
|---------|------|--------|
| `primary` | 주요 강조 | 신규, 추천 |
| `success` | 성공/완료 | 승인됨, 완료 |
| `warning` | 주의 필요 | 검토 중, 만료 예정 |
| `danger` | 오류/위험 | 거절됨, 오류 |
| `info` | 정보 제공 | 업데이트, 안내 |
| `neutral` | 중립 상태 | 기본, 비활성 |


## ✅ Do

- 상태나 카테고리처럼 **맥락이 명확한 짧은 텍스트**에 사용하세요.
- `dot` prop으로 알림 표시나 온라인 상태를 나타내세요.
- 의미에 맞는 variant를 일관되게 사용하세요.

## ❌ Don't

- 긴 문장을 Badge 안에 넣지 마세요. (2~3단어 이내 권장)
- 하나의 항목에 Badge를 3개 이상 붙이지 마세요.
- 색상만으로 의미를 전달하지 마세요. 텍스트도 함께 사용하세요.
- 클릭 가능한 요소에는 Badge 대신 Tag를 사용하세요.


## Badge vs Tag

| | Badge | Tag |
|--|-------|-----|
| 목적 | 상태/수량 표시 | 분류/필터 |
| 인터랙션 | 없음 | 제거 가능 (`onRemove`) |
| 위치 | 다른 요소에 부착 | 독립적으로 나열 |


## 접근성

- 색상만으로 상태를 표현하지 말고 텍스트 또는 아이콘을 함께 사용하세요.
- `dot` 모드는 시각적 표시이므로 `aria-label`로 의미를 보충하세요.
