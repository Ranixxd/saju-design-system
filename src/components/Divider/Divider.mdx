# Divider

콘텐츠 영역을 시각적으로 구분하는 구분선입니다.


## ✅ Do

- 논리적으로 관련 없는 콘텐츠 그룹을 구분할 때 사용하세요.
- `label`로 섹션 제목을 구분선 중앙에 표시할 수 있습니다.
- 세로 구분선(`vertical`)은 인라인 요소나 버튼 그룹 사이에 사용하세요.

## ❌ Don't

- 단순히 여백을 추가하고 싶을 때는 Divider 대신 margin/padding을 사용하세요.
- 모든 항목 사이에 Divider를 넣으면 시각적 노이즈가 생깁니다.
- 카드나 모달 내부 섹션 구분에 Divider를 남용하지 마세요.


## orientation 사용 기준

| 값 | 사용 맥락 |
|----|----------|
| `horizontal` | 섹션 구분, 목록 항목 구분 (기본값) |
| `vertical` | 버튼 그룹, 네비게이션 아이템 사이 |


## 접근성

- `role="separator"`가 자동으로 설정됩니다.
- `vertical` 구분선에는 `aria-orientation="vertical"`이 자동 설정됩니다.
- 순수 장식 목적이라면 `aria-hidden="true"`를 추가하세요.
