# Toggle

즉시 반영되는 On/Off 설정을 위한 스위치 컴포넌트입니다.


## ✅ Do

- **즉시 적용되는 설정**에만 사용하세요. (저장 버튼 없이 바로 반영)
- `label`로 어떤 설정인지 명확히 설명하세요.
- `helperText`로 현재 상태의 의미를 보충 설명하세요. (예: "알림을 받습니다")
- 상태 변경 시 시각적 피드백(색상, 위치)을 즉각 제공하세요.

## ❌ Don't

- 폼 제출이 필요한 경우에는 Toggle 대신 Checkbox를 사용하세요.
- 여러 항목 중 하나를 고르는 용도로 사용하지 마세요. (Radio 사용)
- "켜기/끄기" 외의 의미가 필요하다면 다른 컴포넌트를 고려하세요.
- 레이블 없이 단독으로 사용하지 마세요.


## Toggle vs Checkbox 선택 기준

| 상황 | 권장 컴포넌트 |
|------|-------------|
| 설정이 즉시 반영됨 | Toggle |
| 폼 제출 후 반영됨 | Checkbox |
| 동의/약관 체크 | Checkbox |
| 기능 활성화/비활성화 | Toggle |


## Size 선택 기준

| Size | 사용 맥락 |
|------|----------|
| `sm` | 테이블 내부, 컴팩트 설정 목록 |
| `md` | 기본값, 설정 페이지 |
| `lg` | 중요도 높은 단독 설정 |


## 접근성

- 내부적으로 `role="switch"`를 사용해 스크린리더에 On/Off 상태를 전달합니다.
- `aria-checked`가 자동으로 설정됩니다.
- 키보드 Space 키로 토글할 수 있습니다.
