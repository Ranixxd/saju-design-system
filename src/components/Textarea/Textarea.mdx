# Textarea

여러 줄 텍스트 입력이 필요한 경우에 사용합니다.


## ✅ Do

- 내용이 길어질 것이 예상될 때 사용하세요. (댓글, 메모, 설명 등)
- `label`은 항상 제공하세요.
- 글자 수 제한이 있다면 현재 글자 수와 최대 글자 수를 함께 표시하세요.
- 기본 `resize="vertical"`을 유지해 사용자가 높이를 조절할 수 있게 하세요.

## ❌ Don't

- 한두 줄 입력에 Textarea를 사용하지 마세요. Input을 사용하세요.
- `resize="none"`으로 고정하면 사용자 경험이 나빠집니다. 꼭 필요한 경우에만 사용하세요.
- 자동 높이 조절이 필요하면 CSS `field-sizing: content`를 고려하세요.


## resize 옵션

| 값 | 동작 |
|----|------|
| `vertical` | 세로만 조절 가능 (기본값) |
| `horizontal` | 가로만 조절 가능 |
| `both` | 가로·세로 모두 조절 가능 |
| `none` | 크기 조절 비활성화 |


## 접근성

- `label`을 제공하면 `htmlFor`와 `id`가 자동으로 연결됩니다.
- `errorMessage`가 있으면 `aria-invalid="true"`가 자동 설정됩니다.
- 글자 수 카운터는 `aria-live="polite"`로 스크린리더에 전달하세요.
