# Textarea

여러 줄의 텍스트를 입력받을 때 사용합니다.

---

## ✅ Do

- 한 줄 이상의 입력이 예상되는 경우에 사용하세요. (소개글, 리뷰, 설명)
- `rows`로 **예상 입력 길이에 맞는 초기 높이**를 설정하세요.
- 글자 수 제한이 있다면 `helperText`로 명시하세요. (예: 최대 500자)
- `resize="vertical"` (기본값)을 유지해 사용자가 높이를 조절할 수 있게 하세요.

## ❌ Don't

- 한 줄짜리 입력에는 Textarea 대신 Input을 사용하세요.
- `resize="none"`은 꼭 필요한 레이아웃 제약이 있을 때만 사용하세요. 긴 내용 입력 시 불편합니다.
- 초기 높이(`rows`)를 너무 작게 설정하지 마세요.

---

## rows 가이드

| rows | 적합한 콘텐츠 |
|------|-------------|
| 2~3 | 짧은 메모, 주소 |
| 4~5 | 소개글, 리뷰 |
| 6 이상 | 긴 설명, 본문 작성 |

---

## 접근성

- `label`을 반드시 제공하세요.
- 글자 수 제한은 `maxLength` 속성과 `helperText`를 함께 사용하세요.
- 에러 메시지는 `role="alert"`로 자동 처리됩니다.
