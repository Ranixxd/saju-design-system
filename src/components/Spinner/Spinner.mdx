# Spinner

비동기 작업 진행 중임을 나타내는 로딩 인디케이터입니다.


## ✅ Do

- 응답 시간이 **0.5초 이상** 걸릴 것으로 예상될 때 사용하세요.
- 특정 영역의 로딩이라면 해당 영역 안에 배치하세요.
- 전체 페이지 로딩이라면 화면 중앙에 오버레이와 함께 사용하세요.
- `label`로 스크린리더에 상태를 전달하세요. (예: "데이터를 불러오는 중...")
- Button의 로딩 상태는 `loading` prop을 사용하세요. Spinner를 따로 넣지 마세요.

## ❌ Don't

- 0.3초 이하의 매우 짧은 작업에는 사용하지 마세요. 깜빡임이 더 불편합니다.
- 여러 개의 Spinner를 한 화면에 동시에 보여주지 마세요.
- 로딩이 끝났는데 Spinner를 숨기지 않으면 사용자가 혼란스러워합니다.
- Skeleton UI가 더 적합한 경우(콘텐츠 레이아웃 로딩)에는 Spinner 대신 Skeleton을 사용하세요.


## Variant 선택 기준

| Variant | 사용 맥락 |
|---------|----------|
| `primary` | 기본값, 밝은 배경 위 |
| `white` | 어두운 배경, 컬러 버튼 위 |
| `neutral` | 회색 배경, 중립적 UI |


## Size 선택 기준

| Size | 사용 맥락 |
|------|----------|
| `sm` | 버튼 내부, 인라인 |
| `md` | 기본값, 섹션 로딩 |
| `lg` | 카드, 패널 로딩 |
| `xl` | 전체 페이지 로딩 |


## 접근성

- `role="status"`와 `aria-label`이 자동으로 설정됩니다.
- `label` prop으로 스크린리더에 전달할 텍스트를 커스터마이즈할 수 있습니다.
- 로딩 완료 후에는 결과 영역에 `aria-live="polite"`로 알림을 주세요.
