# Tooltip

요소에 마우스를 올렸을 때 보조 정보를 제공합니다.

---

## ✅ Do

- **아이콘이나 축약된 텍스트**에 추가 설명을 달 때 사용하세요.
- 내용은 **짧고 명확하게** 유지하세요. 1~2문장이 적당합니다.
- `placement`는 화면 가장자리에서 잘리지 않도록 방향을 선택하세요.
- 키보드 포커스 시에도 Tooltip이 표시되도록 하세요 (자동 지원).

## ❌ Don't

- 클릭이 필요한 중요한 정보를 Tooltip에만 넣지 마세요. 모바일에서는 hover가 없습니다.
- Tooltip 안에 링크, 버튼, 폼 요소를 넣지 마세요.
- 이미 텍스트로 명확한 요소에 동일한 내용의 Tooltip을 달지 마세요.
- 긴 문단을 Tooltip에 넣지 마세요. 읽기 어렵고 레이아웃을 벗어날 수 있습니다.

---

## Placement 선택 기준

| Placement | 사용 상황 |
|-----------|---------|
| `top` | 기본값, 대부분의 경우 |
| `bottom` | 상단 네비게이션 아이콘 |
| `left` | 우측 사이드바 요소 |
| `right` | 좌측 사이드바 요소 |

---

## Tooltip vs 다른 컴포넌트 선택 기준

| 상황 | 권장 컴포넌트 |
|------|-------------|
| 짧은 보조 설명 | Tooltip |
| 클릭 후 보여주는 정보 | Popover (추후 구현) |
| 폼 필드 안내 | Input의 `helperText` |
| 즉각적인 피드백 | Toast |

---

## 접근성

- `role="tooltip"` 이 자동 설정됩니다.
- 트리거 요소에 포커스 시 자동으로 Tooltip이 표시됩니다.
- 모바일(터치) 환경에서는 Tooltip이 동작하지 않으므로, 중요한 정보는 다른 방식으로도 제공하세요.
