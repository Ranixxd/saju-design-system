# Tooltip

요소에 마우스를 올렸을 때 추가 설명을 보여주는 컴포넌트입니다.


## ✅ Do

- **아이콘 버튼**처럼 레이블이 없는 요소에 설명을 제공할 때 사용하세요.
- 짧고 보충적인 설명에만 사용하세요. (1~2문장)
- `placement`로 UI가 가려지지 않는 방향을 선택하세요.
- `delay`를 적절히 설정해 의도치 않은 tooltip 노출을 방지하세요. (기본 300ms)

## ❌ Don't

- 필수적인 정보를 Tooltip에만 넣지 마세요. 모바일에서는 hover가 없습니다.
- 긴 설명이 필요하면 Tooltip 대신 별도 도움말 텍스트나 모달을 사용하세요.
- 이미 텍스트 레이블이 있는 버튼에는 불필요한 Tooltip을 붙이지 마세요.
- 클릭 가능한 링크나 버튼을 Tooltip 안에 넣지 마세요.
- `delay`를 0으로 설정하면 너무 민감해집니다.


## placement 선택 기준

| 위치 | 사용 맥락 |
|------|----------|
| `top` | 기본값, 대부분의 상황 |
| `bottom` | 요소가 화면 상단에 있을 때 |
| `left` | 요소가 화면 우측 끝에 있을 때 |
| `right` | 요소가 화면 좌측 끝에 있을 때 |


## 접근성

- Tooltip의 `content`는 트리거 요소의 `aria-describedby`로 연결하세요.
- 키보드 포커스 시에도 Tooltip이 표시되어야 합니다.
- Tooltip이 열렸을 때 `role="tooltip"`이 설정되어야 합니다.
- 모바일에서는 Tooltip이 동작하지 않으므로, 필수 정보는 항상 다른 방식으로도 노출하세요.
